# Ikabay - Marketplace Caribéenne Ultra-Optimisée

## Fonctionnalités Clés
- 🛒 Utilisateur : achat, panier, récompense crypto
- 🛠 Admin : contrôle, analyse, gestion utilisateurs
- 🤝 Partenaires : affiliation, campagnes marketing
- 🧺 Fournisseurs : ventes, inventaire, livraisons

## Déploiement Simplifié
1. Dézippez `ikabay-ultra.zip`
2. Uploadez-le sur Vercel, Netlify ou CodeSandbox
3. Accédez aux pages selon votre rôle :
   - `/admin/dashboard.html`
   - `/partners/partners.html`
   - `/suppliers/suppliers.html`
   - `/users/account.html`

## Objectif
Fournir une plateforme moderne, mobile-first, 100% statique/no-code, scalable et interopérable, centrée sur les DOM-TOM et l'export international, avec système de crypto-récompense intégré.
